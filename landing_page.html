Presentación del Proyecto - Nielsen OCR

Identidad visual para la aplicación de gestión de facturas

Acerca del Proyecto

El proyecto consiste en diseñar la identidad visual para la aplicación Nielsen OCR, incluyendo la creación de un logotipo versátil y elementos gráficos que representen la marca y la función de la aplicación. Este proyecto está dirigido a los empleados de Nielsen para facilitar sus actividades laborales diarias.

Objetivo del Proyecto

El objetivo principal del proyecto es estandarizar los diferentes tipos de tickets de compra o facturas, además de diseñar una interfaz agradable, atractiva e intuitiva para el usuario.

Requerimientos del Proyecto

  • Crear una identidad visual clara y reconocible para la aplicación.
  • Desarrollar un logotipo versátil para diferentes formatos y tamaños.
  • Incorporar elementos gráficos que comuniquen los valores y la misión de la aplicación de manera efectiva.

Público Objetivo

El público objetivo de la aplicación son los empleados de Nielsen que necesitan utilizarla para facilitar sus actividades laborales diarias.

Competencia

La competencia incluye otras aplicaciones similares que pueden ofrecer funcionalidades relacionadas con la gestión de facturas o recibos como Expensify, Receipt Bank o QuickBooks.

Solución de Problemas

El proyecto busca resolver los problemas de comunicación existentes, como la falta de una identidad visual clara para la aplicación y la dificultad para que los usuarios la reconozcan en diferentes plataformas.

Fechas Límites

Se establecerán fechas límite específicas para cada fase del proyecto, con el objetivo de completarlo dentro del plazo de 2 a 3 meses estimado para su ejecución.

Presupuesto del Proyecto

El presupuesto del proyecto consistirá principalmente en la inversión de tiempo del autor en el desarrollo del proyecto.
